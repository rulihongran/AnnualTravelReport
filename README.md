# AnnualTravelReport

旅行记录分析网站项目[WanderWall](https://github.com/rulihongran/WanderWall)中旅行报告部分重构后的版本，可在此基础上更改背景颜色、文字、动图，自定义个人的旅行报告

采用Vue3框架，无缝翻页效果利用Swiper实现
